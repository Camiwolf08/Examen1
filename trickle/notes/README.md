# Sistema de Gestión con Roles de Usuario

## Descripción
Sistema completo de gestión con autenticación, roles de usuarios y dashboard con gráficas estadísticas.

## Características
- Login con efecto cristal (glassmorphism)
- Tema verde claro
- 3 roles de usuario: Superusuario, Administrador y Usuario
- Topbar con notificaciones y perfil
- Sidebar con menú desplegable
- Dashboard con 8 tarjetas estadísticas y gráficas
